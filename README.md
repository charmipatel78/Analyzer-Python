# 📊 NumPy Data Analyzer

## 🧠 Project Idea

This project is built to explore the power of NumPy by performing various operations on arrays in an interactive way.

It allows users to create arrays and apply different transformations, calculations, and analysis techniques.

---

## 🧮 Core Capabilities

🔹 **Array Creation**

* 1D, 2D, and 3D arrays
* Dynamic input from user

🔹 **Mathematical Processing**

* Element-wise addition, subtraction
* Multiplication and division between arrays

🔹 **Array Manipulation**

* Combine arrays (vertical stacking)
* Split arrays into parts

🔹 **Data Exploration**

* Search for specific values
* Sort arrays
* Filter values based on conditions

🔹 **Statistical Analysis**

* Sum, Mean, Median
* Standard Deviation
* Variance

---

## 🔄 Program Flow

1. Create an array
2. Choose operation category
3. Apply operation on array
4. View results instantly

---

## 📦 Library Used

* NumPy

---

## ▶️ Execution

Run the program using:

python "Analyzer.py"
